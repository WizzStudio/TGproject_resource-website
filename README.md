## 项目启动

在本地 shell 或者 git bash 里分别运行如下命令：

- git clone git@github.com:TGclub/TGproject_resource-website.git
- cd TGproject_resource-website
- cnpm install(或者npm install) （本地没有node的先去下载node
- npm run dev
- 在浏览器输入 http://localhost:8090
